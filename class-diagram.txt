Library Management System - Class Relationships

Book Class:
- Attributes: title, author, isbn, publicationYear, isAvailable
- Methods: getters, setters, toString()

Patron Class:
- Attributes: id, name, email, phoneNumber, borrowingHistory, currentlyBorrowed
- Methods: getters, setters, addToBorrowingHistory(), etc.

Library Class:
- Attributes: books, patrons
- Methods: addBook(), removeBook(), searchBooks(), checkoutBook(), returnBook(), etc.

Relationships:
- Library manages Books
- Library manages Patrons
- Patrons borrow Books
